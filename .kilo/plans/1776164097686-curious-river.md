# Plan: Compare Local Code with GitHub Version

## Goal
Verify if the local code on MacBook matches the GitHub repository version.

## Steps
1. Check local git status and current branch
2. Check local recent commits (last 5-10)
3. Fetch latest from GitHub remote
4. Compare local HEAD with GitHub HEAD (main/master branch)
5. Report any differences (commits ahead/behind, uncommitted changes)

## GitHub CLI Commands to Use
- `gh repo view` - Check repository info
- Compare local commits with remote using `git log` and `git fetch`

## Success Criteria
- Show if local matches remote, is ahead, behind, or has diverged
