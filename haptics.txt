HAPTIC STRENGTH GUIDE
=====================

Light Impact:
- Selections, toggles, minor actions
- Menu items, context menu items
- Folder/book selection

Medium Impact:
- Primary actions (create, confirm, move)
- Opening views, confirming operations

Heavy Impact:
- Destructive actions
- Delete confirmations
- Major irreversible changes

Success Notification:
- Successful completion of operations
- Confirmed actions
- Successful saves

Warning Notification:
- Cancellation of operations
- Stopping actions
- Delete button presses (before confirmation)

Error Notification:
- Destructive delete actions (after confirmation)
- Failed operations

Selection Feedback:
- Slider interactions
- Adjusting values
- Precision controls

HOW TO ADJUST HAPTICS
=====================

If you find certain haptics too strong or too weak, you can adjust them:

1. Locate the file and line number from this log
2. Find the HapticManager call
3. Change the haptic type:
   - `.light` → medium or heavy for stronger
   - `.medium` → light or heavy for weaker/stronger
   - `.heavy` → light or medium for weaker
   - `.notification(.success/.warning/.error)` → remove or change type

Examples:
```swift
HapticManager.impact(.light)        // Lightest tap feedback
HapticManager.impact(.medium)       // Medium tap feedback
HapticManager.impact(.heavy)        // Strong tap feedback
HapticManager.selection()           // For sliders and adjustments
HapticManager.notification(.success)  // Success completion
HapticManager.notification(.warning)  // Warning/cancellation
HapticManager.notification(.error)    // Error/destructive action
```

To remove a haptic:
```swift
// Before:
Button(action: {
    HapticManager.impact(.light)
    doSomething()
})

// After:
Button(action: {
    doSomething()
})
```
